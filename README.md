# AI Job Market Analysis Dashboard

## Project Overview
This Power BI dashboard analyzes AI job demand, salary trends, experience levels, and remote work patterns.

## Business Objective
To identify hiring patterns and generate insights that support workforce planning and career strategy decisions.

## Tools Used
- Power BI Desktop
- DAX
- Microsoft Excel
- CSV Dataset

## Key Metrics
- Total Job Openings
- Average Salary
- Experience Level Distribution
- Remote Job Count

## Key Insights
- Mid-level roles dominate hiring demand
- Senior roles earn higher salary packages
- Data Analyst and Machine Learning Engineer are high-demand roles
- Remote opportunities increase with experience level
